# CORTEX 自动测试-评估-迭代框架

## 自动化流程

```
迭代循环 (50次)
├── Test Phase (测试阶段)
│   ├── 代码静态分析
│   ├── 编译检查
│   ├── 逻辑验证
│   └── 问题检测
├── Evaluate Phase (评估阶段)
│   ├── 问题分类
│   ├── 严重度评级
│   └── 修复优先级
├── Iterate Phase (迭代阶段)
│   ├── 实施修复
│   ├── 验证修复
│   └── 记录变更
└── Report Phase (报告阶段)
    └── 生成迭代报告
```
